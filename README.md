2.段落
直接输入文本形成段落，段落之间用一个空行分隔。

3.换行
在行末添加两个或多个空格，然后按回车
强制换行：第一行<br>第二行


4.强调
加粗：用 ** 或 __ 包裹文字。
斜体：用 * 或 _ 包裹文字。
加粗斜体：用 *** 或 ___ 包裹文字。

    **加粗**
    *斜体*
    ***加粗斜体***

5.列表
无序列表：使用 -、+ 或 * 创建无序列表。
有序列表：使用数字加 . 来创建有序列表。
- > 会出现一个带阴影的列表

    - 无序列表项 1
    - 无序列表项 2
    - 无序列表项 3

    1. 有序列表项 1
    2. 有序列表项 2
    3. 有序列表项 3

6.引用
使用 > 表示引用。

    > 这是一个引用段落。
    > > 嵌套引用。


7.使用 [链接文本](链接地址) 来创建链接。

    [Google](https://www.google.com)

8. 图片
使用 ![图片描述](图片地址) 插入图片。

    ![示例图片](https://example.com/image.jpg)

9. 代码
行内代码：使用反引号 ` 包裹代码片段。
代码块：使用三个反引号 ``` 包裹代码块或缩进四个空格。，可以指定语言。

    这是 `行内代码` 的示例。
    第一行可以指定语言
    ```python
    def hello_world():
        print("Hello, World!")
    ```
    注意顶格


10.表格
使用 `|` 来分隔列，使用 `-` 来分隔表头和表体。

    | 表头1 | 表头2 | 表头3 |
    | ----- | ----- | ----- |
    | 内容1 | 内容2 | 内容3 |
    | 内容4 | 内容5 | 内容6 |

11. 水平线
使用三个或更多的 -、*、_ 来创建水平线。

12. 任务列表
使用 - [ ] 表示未完成任务，使用 - [x] 表示已完成任务。

13. 脚注
使用 [^注释] 在文本中创建脚注，并在文档底部添加脚注内容。

    这是一个有脚注的文本[^1]。
    [^1]: 这是脚注内容。 必须得是两个单词及以上。

14.链接
Markdown 会自动将 URLs 转换为链接，甚至不需要 [文本](链接) 的语法。

15.字体颜色与大小（HTML 支持）
Markdown 本身不支持改变字体颜色或大小，但可以嵌入 HTML。

    <span style="color: red;">红色文字</span>
    <span style="font-size: 20px;">大字号文字</span>

16.下划线
Markdown 并不直接支持下划线，但可以使用 HTML 标签 <u>。

   <u>带下划线的文字</u>

17.注释
注释使用 <!-- -->，可以在文档中添加不会显示的内容。
    <!-- 这是注释，不会在页面上显示 -->

18.表情
一些 Markdown 解析器支持 GitHub 风格的表情符号（emoji），可以使用 :emoji_name:。

    :smile: :rocket: :thumbsup:

20.空格
markdown只能识别一个空格
插入一个空格 (non-breaking space) (也可以用作空行)
　　　　&nbsp;    或    &#160;     或      &#xA0;

插入两个空格 (en space)
　　　　&ensp;     或    &#8194;   或      &#x2002;

插入四个空格 (em space)
　　　　&emsp;    或    &#8195;   或      &#x2003;

插入细空格 (thin space)
　　　　&thinsp;   或     &#8201;  或      &#x2009;

插入空行：
　　　　&nbsp;或 <br/> 都可以，
